 Основные проекты:


Heart Attack Risk Prediction: 

Классический ML: выявление ключевых факторов риска (Feature Importance) сердечных заболеваний.

Credit Card Fraud Detection:

Борьба с экстремальным дисбалансом классов (99/1), максимизация Recall для минимизации потерь.

Twitter Sentiment Analysis: 

Анализ тональности постов, предобработка текста и работа с классическими ансамблями.

UCI ML Drug Review: 

NLP-проект: прогнозирование рейтинга лекарства по тексту отзыва пациента.

Animal Faces Classificator: 

Computer Vision: сверточные нейросети (CNN) для распознавания видов животных.

Face Expression Recognition:

Deep Learning: классификация эмоций человека по изображениям лиц.

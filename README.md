 Основные проекты:

Credit Card Fraud Detection (Finance & Classic ML)

Задача: Обнаружение мошеннических транзакций в условиях экстремального дисбаланса классов (99.8% vs 0.2%).

Решение: Настройка весов классов (class_weight), использование Random Forest и Gradient Boosting.

Результат: Максимизация Recall для минимизации финансовых потерь бизнеса.

Heart Attack Risk Prediction

Задача: Предсказание риска сердечного приступа на основе медицинских показателей.

Ключевой инсайт: Анализ важности признаков (Feature Importance) для выявления критических факторов риска.

Face Expression & Animal Faces (Computer Vision)

Задача: Классификация эмоций человека и видов животных по изображениям.

Стек: Сверточные нейронные сети (CNN), аугментация данных для повышения точности.

UCI ML Drug Review & Twitter Sentiment (NLP)

Задача: Анализ тональности отзывов на лекарства и постов в Twitter.

Методы: Предобработка текста (лемматизация, очистка), классификация текстов с помощью классических ансамблей и нейросетей.
